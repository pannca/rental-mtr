# Aplikasi Rental Motor (PHP)

Aplikasi sederhana berbasis PHP untuk menghitung total biaya sewa motor berdasarkan:
- Jenis motor
- Lama sewa (per hari)
- Status member atau non-member

Aplikasi ini juga menerapkan:
- Diskon untuk member
- Pajak tetap
